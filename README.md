# Spire Archive

The complete database for [Slay the Spire](https://store.steampowered.com/app/646570/Slay_the_Spire/) — browse cards, relics, potions, characters, monsters, events, powers, and keywords.

**Live site:** [spire-archive.com](https://spire-archive.com)

## Stack

- **Astro** SSR with `@astrojs/node` adapter
- **Tailwind CSS v4**
- **React islands** for interactive filters
- Game data in `./data/` (generated by `./parsers/`)

## Local dev

```bash
npm install
npm run dev
# http://localhost:4321
```

## API

All endpoints return JSON with `Cache-Control: public, max-age=3600`.

| Endpoint | Filters |
|----------|---------|
| `GET /api/cards` | `q`, `color`, `type`, `rarity`, `cost`, `offset`, `limit` |
| `GET /api/relics` | `q`, `tier`, `color`, `offset`, `limit` |
| `GET /api/potions` | `q`, `rarity`, `offset`, `limit` |
| `GET /api/monsters` | `q`, `act`, `type`, `offset`, `limit` |
| `GET /api/events` | `q`, `act`, `offset`, `limit` |
| `GET /api/powers` | `q`, `type`, `offset`, `limit` |
| `GET /api/search` | `q`, `limit` |

Detail endpoints: `GET /api/{resource}/:id`

## Deployment

Deploys automatically to Azure Container Apps on push to `main` via GitHub Actions.

Manual deploy: `./infra/deploy.sh`
